# RFID Door Automation System 🚪

This project is an IoT-based door automation system using ESP32, RFID, LCD, and Servo Motor.

## 🔧 Features
- RFID card authentication
- Automatic door opening using servo
- LCD display status messages
- Buzzer alerts for access granted/denied
- LED indication

## 🧰 Components Used
- ESP32
- MFRC522 RFID Module
- Servo Motor
- I2C LCD Display
- Buzzer
- LED

## ⚙️ How It Works
When a valid RFID card is scanned, the door opens automatically. If the card is invalid, access is denied with buzzer alert.
